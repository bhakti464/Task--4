# Task 4 – SQL Joins (Chinook Database)

## Overview

This project demonstrates the use of **SQL JOIN operations (INNER JOIN and LEFT JOIN)** to analyze business data using the **Chinook SQLite database**.

## Tools Used

* DB Browser for SQLite
* SQLite (Chinook Database)
* Excel (for exporting CSV output)
* Notepad (for insights documentation)

## Tasks Performed

* INNER JOIN to combine customers and invoices
* LEFT JOIN to find customers with no orders
* Calculated revenue per product
* Analyzed revenue by category (genre)

## Files Included

* `joins_queries.sql` – SQL queries
* `joined_output.csv` – Query results
* `insights.txt` – Business insights

## Learning Outcome

Learned how to use SQL joins to answer real-world business questions.
